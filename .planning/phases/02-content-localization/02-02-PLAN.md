# Plan 02-02: Add Korean Translations

## Why This Matters

Korean translations allow the website to reach Korean-speaking clients, a key target market for the freelance business.

## Technical Notes

**Korean translation guidelines:**
- Use formal/polite Korean (존댓말) for business context
- Keep brand name "WebCraft" in English
- Keep technology names in English
- Adapt phrases naturally rather than literal translation

**Example translations:**

  "nav": {

    "services": "서비스",
