# Plan 02-01: Extract Hardcoded Text to Translation Files

## Overview

Extract all hardcoded UI text from React components into the English translation file, then update components to use the `useTranslation` hook.

## Why This Matters

This is the foundation for bilingual support. Once all text is in translation files, adding Korean translations becomes straightforward.

## Technical Notes

**Translation file structure:**
```json
{
  "nav": { "home": "Home", "services": "Services", ... },
  "hero": { "title": "Transform Your Vision Into", ... },
  "services": { "title": "Services I Offer", "items": [...] },
  ...
}
```

**Component pattern:**
```tsx
import { useTranslation } from 'react-i18next';

export function Component() {
  const { t } = useTranslation();
  return <h1>{t('section.title')}</h1>;
}
```

**For arrays, use `returnObjects: true`:**
```tsx
const items = t('services.items', { returnObjects: true }) as ServiceItem[];
```
