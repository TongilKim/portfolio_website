# Plan 02-03: Replace Placeholder Business Data

## Overview

Replace demo/placeholder content with real business information. This includes contact details, portfolio items, and business-specific text.

## Why This Matters

Placeholder content (fake emails, phone numbers, generic projects) undermines credibility. Real data makes the site production-ready.

## Tasks

- [ ] **Task 1**: Gather real business information
  - Confirm actual email address for contact
  - Confirm phone number (or remove if not providing)
  - Confirm location/service area
  - Confirm social media links (or remove placeholders)
  - Files: None (information gathering)

- [ ] **Task 2**: Update contact information in translations
  - Replace `hello@webcraft.com` with real email
  - Replace `+1 (555) 123-4567` with real phone or remove
  - Update location text as appropriate
  - Update in both EN and KO translation files
  - Files: `src/locales/en/translation.json`, `src/locales/ko/translation.json`

- [ ] **Task 3**: Update or remove social media links
  - Update Twitter/X link or remove
  - Update LinkedIn link or remove
  - Update GitHub link or remove
  - Files: `Footer.tsx`, `FooterV2.tsx`

- [ ] **Task 4**: Review and update portfolio projects
  - Decide: Keep placeholder projects or add real ones
  - Update project titles/descriptions if using real projects
  - Update project images if using real screenshots
  - Files: `Portfolio.tsx`, translation files

- [ ] **Task 5**: Review stats accuracy
  - Update "50+ Projects" if not accurate
  - Update "5+ Years Experience" if not accurate
  - Update or remove stats that don't apply
  - Files: `Hero.tsx`, translation files

- [ ] **Task 6**: Update brand name if needed
  - If not using "WebCraft", update throughout
  - Update in Header, Footer, meta tags
  - Files: Multiple component files, translation files

- [ ] **Task 7**: Review all placeholder text
  - Search for "example", "placeholder", "lorem" text
  - Replace form placeholder examples with appropriate text
  - Files: All components

- [ ] **Task 8**: Final review and verification
  - Review entire site for any remaining placeholder content
  - Verify all contact methods work
  - Test form displays correctly
  - Files: None (verification)

## Acceptance Criteria

- [ ] No fake contact information (emails, phones)
- [ ] Social links point to real profiles or are removed
- [ ] Stats are accurate or appropriately generic
- [ ] Portfolio represents actual or representative work
- [ ] Brand name is finalized
- [ ] No "example.com" or similar placeholder domains
