# Plan 06-02: SEO Meta Tags and Open Graph

## Summary

Add comprehensive SEO meta tags, Open Graph, and Twitter Card markup to index.html for better search visibility and social sharing.

## Detailed Steps

### Step 1: Update index.html with meta tags

File: `index.html`

Replace current minimal head with comprehensive SEO-friendly markup:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

    <!-- Primary Meta Tags -->
    <title>John Doe | Freelance Web Developer - Custom Websites & Web Apps</title>
    <meta name="title" content="John Doe | Freelance Web Developer - Custom Websites & Web Apps" />
    <meta name="description" content="Professional freelance web developer specializing in custom websites, web applications, and e-commerce solutions. Get a quote for your next project." />
    <meta name="keywords" content="web developer, freelance, custom websites, web applications, React, responsive design, 웹 개발자, 프리랜서" />
    <meta name="author" content="John Doe" />
    <meta name="robots" content="index, follow" />

    <!-- Open Graph / Facebook -->
    <meta property="og:type" content="website" />
    <meta property="og:url" content="https://yoursite.com/" />
    <meta property="og:title" content="John Doe | Freelance Web Developer" />
    <meta property="og:description" content="Professional freelance web developer specializing in custom websites, web applications, and e-commerce solutions." />
    <meta property="og:image" content="https://yoursite.com/og-image.jpg" />
    <meta property="og:locale" content="en_US" />
    <meta property="og:locale:alternate" content="ko_KR" />

    <!-- Twitter -->
    <meta name="twitter:card" content="summary_large_image" />
    <meta name="twitter:url" content="https://yoursite.com/" />
    <meta name="twitter:title" content="John Doe | Freelance Web Developer" />
    <meta name="twitter:description" content="Professional freelance web developer specializing in custom websites, web applications, and e-commerce solutions." />
    <meta name="twitter:image" content="https://yoursite.com/og-image.jpg" />

    <!-- Favicon -->
    <link rel="icon" type="image/svg+xml" href="/favicon.svg" />

    <!-- Theme Color -->
    <meta name="theme-color" content="#0284c7" />
  </head>

  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.tsx"></script>
  </body>
</html>
```

### Step 2: Create placeholder favicon

File: `public/favicon.svg`

Create a simple SVG favicon with initials:

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <rect width="100" height="100" rx="20" fill="#0284c7"/>
  <text x="50" y="65" font-family="system-ui, sans-serif" font-size="45" font-weight="bold" fill="white" text-anchor="middle">JD</text>
</svg>
```

## Notes

- Theme color matches the primary blue from Tailwind (sky-600)
